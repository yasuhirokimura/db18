License Change

Starting with the 18.1 releases, Berkeley DB products are licensed in two ways depending on the source of the package. Packages downloaded from the Oracle Technology Network (OTN) are licensed under the GNU AFFERO GENERAL PUBLIC LICENSE (AGPL), version 3. This license is published by the Free Software Foundation (FSF) and approved by the Open Source Initiative. Packages downloaded from the Oracle Software Data Cloud (OSDC) are licensed under the Oracle Master Agreement (OMA). Additionally, starting with the 18.1 releases, example code included in the packages is in all cases licensed under the Universal Permissive License (UPL).

Please review the terms of the appropriate license(s) to ensure compliance before upgrading to the 18.1 release.
